# NAMOAROGYA Icons Directory

This directory should contain PWA icons in the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Generating Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Photoshop/Figma/Sketch

## Icon Design Guidelines

- Use the NAMOAROGYA logo with the Activity icon
- Primary color: #0891b2 (medical teal)
- Background: white or transparent
- Ensure icons are clear at all sizes
- Follow PWA maskable icon guidelines for 192x192 and 512x512
